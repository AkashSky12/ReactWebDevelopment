CSS: .grid-container { display: grid; grid-template-columns: 1fr 2fr; grid-template-row: 1fr 1fr; gap: 10px; } - Template rows/columns can be created using fr, px, rem, grid-template - Grid created using px/rem are not responsive. i,e. if we change the width and height of screen, then grid doesn't change - Other way to represent grid-template-row/column: Ex: grid-template: 100px 200px / 400px 800px; //first part defined is rows and the second is column - We can use auto to setup the grid: Ex: .grid-container { display: grid; grid-template-columns: 200px auto; //setting column auto takes up 100% width of webpage grid-template-row: 100px auto; //setting row auto makes sure that the space is set for content to fit } Reference: https://appbrewery.github.io/grid-sizing/
